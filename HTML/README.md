1-Consulte y explique que significa a nivel de uso del internet: 
a. Servidor DNS 
b. IP 
c. Protocolo 
d. Nombre de dominio 
e. Directorio y nombre de archivo 

R=
 a- Un servidor DNS (Sistema de Nombres de Dominio) funciona como una agenda telefónica de Internet. Su tarea es traducir los nombres de dominio como google.com a direcciones IP que las computadoras entienden

b-Es un identificador único para cada dispositivo conectado a Internet
Ejemplo: 192.168.1.1
 Permite que los dispositivos se encuentren y se comuniquen entre sí

c-Es un conjunto de reglas que definen cómo se comunican los dispositivos en una red
Ejemplo: HTTP, HTTPS, FTP
Garantiza que la información se envíe y reciba correctamente

d-Es el nombre fácil de recordar que representa una dirección IP
Ejemplo: www.youtube.com
 Es lo que escribimos en el navegador para acceder a un sitio web

e-Es la ruta específica dentro de un sitio web que indica dónde está un recurso.
Ejemplo: www.ejemplo.com/blog/articulo.html
blog/ - directorio
articulo.html - archivo
 Sirve para organizar y acceder a contenido dentro de un sitio web

2. Mencionamos dos roles importantes en el desarrollo de un producto web, pero hay 
otros roles involucrados, consulte y defina qué hace cada uno de los siguientes 
roles: 
a. Diseñador UI/UX 
b. Cloud Manager / Ingeniero de infraestructura en la nube 
c. Administrador de base de datos 
d. Project Manager / Director del Proyecto 

R=
a-Diseñador UI/UX
Se encarga de la apariencia (UI) y la experiencia del usuario (UX) y diseña interfaces atractivas y fáciles de usar

b-Cloud Manager / Ingeniero de infraestructura en la nube
Administra los servidores, redes y servicios en la nube (como AWS o Azure) y se asegura de que el sistema sea escalable, seguro y esté siempre disponible

c-Administrador de base de datos (DBA)
Gestiona las bases de datos y se encarga de almacenar, organizar, proteger y optimizar los datos

d-Project Manager / Director del Proyecto
Coordina al equipo y supervisa el desarrollo del proyecto y Controla tiempos, recursos y objetivos
