Semana 4:Modelo de caja y CSS avanzado

El modelo de caja y el display

Cada elemento HTML es tratado por el navegador como una caja rectangular con height, width, margin, padding y border. Gracias a la regla box-sizing: border-box que hemos aplicado en el CSS, ajustar las dimensiones es mucho más predecible.

Los navegadores web aplican valores por defecto a los elementos. Este párrafo es un elemento de bloque (display: block), mientras que las palabras en color fucsia son en línea (display: inline)

Jugando con fondos (Background)

El logotipo de CSS de la izquierda utiliza background-size: contain. Esto garantiza que la imagen aumente o se reduzca proporcionalmente sin ser recortada, adaptándose exactamente al contenedor de 150x150px.
